# InfoReport Studio

An interactive writing app for **Year 5 Information Reports** on renewable resources.

## Features

- **Left panel** — Colour-coded exemplar on *Kryptonite* (an uncommon topic students won't copy) with clickable highlights and pop-up explanations
- **Middle panel** — Section-by-section writing guide with tips, sentence starters, and checklists
- **Right panel** — Structured text editor with word counter
- **Purple sidebar** — Section navigation, progress ring, Save, Copy, Download, Print, and Clear
- **Auto-save** — Work is saved to your browser automatically
- **Dark mode** — Toggle in the top bar; preference is remembered

## Quick Start

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually `http://localhost:5173`).

## Build for classroom use

```bash
npm run build
npm run preview
```

Or deploy the `dist` folder to any static host.

## Report Structure

| Section | Focus |
|---------|-------|
| Heading | Name of renewable resource |
| Introduction | What is it? |
| Paragraph 1 | How it works |
| Paragraph 2 | Advantages |
| Paragraph 3 | Interesting facts / use in Australia |
| Conclusion | Brief summary |
